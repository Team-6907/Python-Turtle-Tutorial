## 🎯 如何填空

看到这样的代码：

```python
# TODO: 设置窗口大小为 800 x 600
screen.setup(______, ______)
```

你需要把 `______` 替换成正确的值：

```python
screen.setup(800, 600)
```

## ❓ 常见问题

**Q: 运行时报错怎么办？**
A: 检查是否有拼写错误，特别是大小写

**Q: 窗口无法关闭？**
A: 点击窗口内部，或按 Ctrl+C

**Q: 按键没反应？**
A: 确保鼠标点击了游戏窗口（让窗口获得焦点）
